This project performs customer segmentation using K-Means clustering to analyse customer behaviour and churn patterns.

Data cleaning and preprocessing.
Train-test split and scaling (StandardScaler).
Elbow Method used to select optimal clusters.
K-Means clustering with 4 clusters

Customers were grouped into 4 clusters.
Cluster 1 shows relatively higher churn.
Other clusters show moderate churn behaviour.

The final dataset includes a cluster column:processed_data_with_clusters.csv.

  Python, Google Colab, Pandas, Scikit-learn, Matplotlib were used.
